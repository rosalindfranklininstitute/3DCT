To create an standalone executable that does not require complicate python installations run the following in the command line

pyinstaller TDCT_main.py --name "3DCorrelationToolbox-RFI" -i "icons\3DCT_icon.ico" --windowed --hidden-import="skimage.filters.rank.core_cy_3d" --add-data "TDCT_correlation.ui;." --add-data "TDCT_main.ui;." --add-data "icons_rc.py;."

It should create a new file or folder in /dist folder with the executable 3DCorrelationToolbox-RFI

To debug the executable, best use without the --windowed option
pyinstaller TDCT_main.py --name "3DCorrelationToolbox-RFI" -i "icons\3DCT_icon.ico" --hidden-import="skimage.filters.rank.core_cy_3d" --add-data "TDCT_correlation.ui;." --add-data "TDCT_main.ui;." --add-data "icons_rc.py;."

2021/10/11 IMPORTANT
Reikna - pyinstaler issues. Pyinstaller exectutable cannot find .mako files that are needed for running Reikna GPU accelerations.
Solution found: mondify .spec file manually. Spec file is created when running pyinstaller.
Modify to include the following lines at start:
`
import importlib
reikna_module_path = importlib.util.find_spec('reikna').submodule_search_locations[0]
`

... and modify [datas] to include the following.
` datas=[ (reikna_module_path , 'reikna' )], `



Create a single executable file (Not recommended, It can fail to run on some systems)
pyinstaller TDCT_main.py --name "3DCorrelationToolboxHKHF-RFI-single" -i "icons\3DCT_icon.ico" --windowed --onefile --hidden-import="skimage.filters.rank.core_cy_3d" --add-data "TDCT_correlation.ui;." --add-data "TDCT_main.ui;." --add-data "icons_rc.py;."
